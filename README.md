|         Item          | Value                                           |
| :-------------------: | :---------------------------------------------- |
|          OS           | ArchLinux                                       |
|          WM           | i3-gaps                                         |
|         Shell         | zsh _([zimfw](https://github.com/zimfw/zimfw))_ |
|          DM           | lightdm, SLIM                                   |
|         Font          | iosevka, iosevka nerd font                      |
|      Compositor       | picom(compton fork)                             |
